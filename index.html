⚙️ Firebase Setup — One Time Only

Real multiplayer QR joining requires a free Firebase project (5 min setup):

  1. Go to console.firebase.google.com
  2. Add Project → name it anything → skip Analytics
  3. Left menu: Build → Realtime Database → Create Database
  4. Choose region → Start in test mode → Enable
  5. Go to Project Settings ⚙️ → Your Apps → </> Web
  6. Register app → copy the firebaseConfig object values
  7. Open this HTML file in any text editor, paste values into the config section near the top
  8. Save and reopen — players can now scan and join! 🎉
